** If you want to modify this code and run it with your own account:

1. Create a file named '.env.twitter' in your home directory '/Users/username/.env.twitter'

2. Populate the file with the following credentials:

TWITTER_CONSUMER_KEY= ""
TWITTER_CONSUMER_SECRET= ""
TWITTER_ACCESS_TOKEN= ""
TWITTER_ACCESS_TOKEN_SECRET= ""
TWITTER_WEBHOOK_ENV= ""
PLOTLY_API= ""
PLOTLY_USER= ""
